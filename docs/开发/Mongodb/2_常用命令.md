# mongodb常用命令



| 命令                | 说明                                |
| ------------------- | ----------------------------------- |
| `show dbs`          | 显示当前 MongoDB 实例中的所有数据库 |
| `use <dbname>`      | 切换到指定数据库                    |
| `db`                | 显示当前数据库名称                  |
| `cls`               | 清除终端屏幕                        |
| `show collections`  | 显示当前数据库中的所有集合          |
| `db.dropDatabase()` | 删除当前数据库                      |
| `exit`              | 退出 mongosh 会话                   |
